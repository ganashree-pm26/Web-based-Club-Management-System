# Web-Based Club Management System

## Project Overview
The Web-Based Club Management System is a centralized platform designed to manage college clubs, events, participants, coordinators, budgets, expenditures, and feedback efficiently.  
It provides role-based access to administrators and coordinators, ensuring secure and organized handling of club activities.

This project is developed as part of a **DBMS mini-project**, focusing on database design, data retrieval, security, and real-world application integration.

---

## Key Features
- Club and Event Management
- Participant Registration and Attendance Tracking
- Coordinator and Member Management
- Budget Allocation and Expenditure Tracking
- Sponsor Management
- Feedback Collection and Analysis
- Interactive Admin Dashboard
- Role-Based Authentication and Authorization

---

## Innovative Components
1. **AI-Based Feedback Insight Generation**  
   Automatically analyzes participant feedback to generate sentiment-based insights, helping administrators understand event success and areas of improvement.

2. **Data Analytics Dashboard**  
   Visual representation of participation trends, budget utilization, and event performance for informed decision-making.

---

## User Roles
### Admin
- Create and manage clubs
- Create coordinators and assign roles
- Manage events and budgets
- Approve expenditures
- View analytics and reports
- Analyze participant feedback

### Coordinator
- Manage assigned events
- Register participants
- Track attendance
- Submit expenditure details

## Security & Validation
- Role-based access control (Admin / Coordinator)
- Secure login authentication
- Server-side form validation
- Input constraints enforced through database schema
- Restricted access to admin-only modules
